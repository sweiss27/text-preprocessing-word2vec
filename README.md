# Text Data Preprocessing and Word Embedding (Word2Vec)

This project preprocesses a raw text dataset from the NLTK Gutenberg corpus (austen-emma.txt), performs exploratory data analysis (EDA) on word frequencies, and trains a Word2Vec embedding model.

## What’s included
- `text_preprocessing_emma.py` — full preprocessing + EDA + Word2Vec
- Notebook (`.ipynb`) — same workflow with charts

## How to run
1. Install dependencies:
   pip install nltk pandas matplotlib seaborn gensim scikit-learn wordcloud

2. Run:
   python text_preprocessing_emma.py
